## 자주 사용하는 명령어
- 새로운 포스트를 생성하고자 할 경우
```
hugo new posts/my-first-post.md
```
- 로컬서버에서 테스트 하고자 할 경우
```
hugo server -D
```
- 최종적으로 적용된 파일들을 만들고자 할 경우
```
hugo -D
```
